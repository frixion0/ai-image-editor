Enhance Image API

Automatically improve image quality using AI. This endpoint sharpens, denoises, and improves the overall visual appeal of an image.

HTTP Request

POST /api/enhance

Example Code

const baseUrl = document.getElementById('baseUrl').value;
const photoDataUri = 'data:image/jpeg;base64,...'; // Your base64 encoded image

fetch(`${baseUrl}/api/enhance`, {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ photoDataUri }),
})
.then(response => {
  if (!response.ok) {
    return response.json().then(err => { throw new Error(err.error) });
  }
  return response.json();
})
.then(data => console.log('Success:', data.enhancedPhotoDataUri))
.catch(error => console.error('Error:', error));

AI Manipulation API

Manipulate an image based on natural language instructions (e.g., "make the sky blue").

HTTP Request

POST /api/manipulate

Example Code

const baseUrl = document.getElementById('baseUrl').value;
const photoDataUri = 'data:image/jpeg;base64,...'; // Your base64 encoded image
const instructions = 'Make the sky blue';

fetch(`${baseUrl}/api/manipulate`, {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ photoDataUri, instructions }),
})
.then(response => {
  if (!response.ok) {
    return response.json().then(err => { throw new Error(err.error) });
  }
  return response.json();
})
.then(data => console.log('Success:', data.editedPhotoDataUri))
.catch(error => console.error('Error:', error));
